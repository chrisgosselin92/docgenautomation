# LegalDocGenerator

Version 0.0.1

A local GUI tool for:
- Managing clients
- Managing document variables
- Generating Word documents from templates
- Bulk-importing variables from templates
- Exporting client data to Excel

## Requirements
- Python 3.11+
- SQLite (bundled with Python)

python run.py
